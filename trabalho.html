Entendendo a magia negra

A magia negra é sustentada por forças e espíritos grandiosos; portanto, é importante saber no que você está entrando antes de começar a lançar feitiços ou conjurar pragas — de outro modo, seu próprio ser poderia acabar prejudicado.

Quais os resultados que você deseja alcançar? Qual problema está incomodando-o tanto ao ponto de você desejar usar magia negra para corrigi-lo? A magia negra é considerada uma arte escura pelo fato de você usá-la para realizar os seus desejos através do controle sobre outro alguém.Se seu objetivo é ajudar outros ou criar alguma forma de paz e justiça, experimente magia branca. Se quiser praticar magia para algum ganho pessoal, as artes negras serão convenientes. Eis alguns motivos pelos quais as pessoas usam magia negra:

  • Para manter alguém paralisado. Se alguém estiver prejudicando-o e você desejar parar essa pessoa, é possível usar uma magia restritiva para negar as ações desse sujeito.
  • Para atrair alguém. Magias de amor estão entre as mais populares quando se fala de feitiçaria negra
  • Para alcançar a imortalidade ou melhorar a saúde.
  • Para se comunicar com os mortos.

Conheça os básicos sobre rituais de magia negra. O ritual usado para fazer a magia negra dependerá do resultado desejado. Diferentes rituais geram diferentes efeitos: por exemplo, a magia relativa ao ganho de fortuna difere daquela que busca ressuscitar os mortos. A maioria dos rituais envolve os seguintes elementos:

  • Um campo é escolhido para a realização da maldição ou feitiço.
  • Um círculo é desenhado no campo de feitiçaria. Um pentagrama é desenhado dentro do círculo. Esse é o círculo de poder.
  • Velas, ervas, cristais, amuletos e outros materiais são empregados para atrair os espíritos.
  • Palavras de poder (relevantes ao resultado desejado) são repetidas três vezes.
